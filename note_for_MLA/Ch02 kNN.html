np.tile函数说明¶

tile(A, reps)
repsA

  • Anp.array
  • reps
  • tileAreps
  • (a = np.array([[1,2], [3,4]]))
    • np.tile(a, 2)
      Output: array([[1,2,1,2], [3,4,3,4]]) -> 2x4
    • np.tile(a, (2,1))
      Output: array([[1,2], [3,4], [1,2], [3,4]]) -> 4x2
